# 07. Fluxo de Defeitos

## Registro
Defeitos serão registrados na aba `Issues`.

## Conteúdo Mínimo
- título
- ambiente
- pré-condição
- passos para reproduzir
- resultado esperado
- resultado obtido
- severidade
- prioridade
- evidência

## Fluxo Sugerido
- `Open`
- `Triaged`
- `In Progress`
- `Retest`
- `Closed`
